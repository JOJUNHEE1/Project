#🌍 Landmarks Travel Blog

아시아, 유럽, 북미의 대표 랜드마크들을 모아 소개하는 개인 여행 블로그 프로젝트입니다.

각 랜드마크마다 사진, 주요 포인트, 간단 설명을 제공하고

사용자가 필터 · 검색 · 랜덤 추천 기능을 통해 직관적으로 탐색할 수 있도록 구성했습니다.

⸻

##✨ Features

###🌏 대륙 필터링 (3개 대륙 지원)
	
  •	Asia (아시아)
  
•	Europe (유럽)
	
  •	North America (북미)

선택하면 해당 대륙의 랜드마크만 빠르게 확인할 수 있습니다.

###🖼 카드형 랜드마크 목록
	
  •	썸네일 사진
	
  •	위치(도시/국가)
	
  •	한 줄 요약 정보

깔끔한 그리드 UI로 어떤 랜드마크인지 한눈에 파악 가능합니다.
